⚠️ Setup Required

To make this app work across all phones:

  1. Go to jsonbin.io
  2. Create a free account
  3. Create a new bin
  4. Copy your bin ID and API key
  5. Replace YOUR_BIN_ID and YOUR_API_KEY in the code

Once set up, all phones will see the same property listings!
